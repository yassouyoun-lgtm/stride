# Stride 

Stride is a shoe shopping website built with **HTML**, **CSS**, and a touch of **JavaScript**. It showcases a clean, modern shopping experience with a bold color palette of **lime, white, and black**.

## Features

-  **Home Page** – Landing page introducing the Stride brand and featured shoes
-  **Product Detail Pages** – Dedicated pages with more information about each shoe (description, price, sizing, etc.)
-  **Contact Page** – A form for users to reach out with questions or feedback
-  **Sign Up Page** – A registration page for new users to create an account
-  **Custom Styling** – Consistent lime/white/black theme across all pages
-  **Interactive Elements** – Small JavaScript enhancements (e.g. form validation, menu toggles, etc.)

## Tech Stack

| Technology | Purpose |
|------------|---------|
| HTML       | Page structure and content |
| CSS        | Styling, layout, and color theme |
| JavaScript | Basic interactivity |

## Color Palette

- 🟢 Lime — accents, buttons, highlights
- ⚪ White — backgrounds, spacing
- ⚫ Black — text, contrast elements

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your browser
3. Navigate through the site using the nav bar to explore products, sign up, or contact us
